# Toy App

Rails チュートリアル第2章で作成した練習用アプリケーションです。

## デモ

https://toy-app-hwzs.onrender.com

## 機能

- ユーザーの CRUD（作成・表示・更新・削除）
- マイクロポストの CRUD
- ユーザーとマイクロポストの関連付け（1対多）
- バリデーション（名前・メールの必須、投稿の140文字制限）

## 技術スタック

- Ruby 3.1.4
- Rails 7.1.6
- SQLite3（開発・テスト）
- PostgreSQL（本番）
- Puma（Web サーバー）
- Render（ホスティング）
